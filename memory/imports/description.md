# Описание приложения для скриншотов

## Общее назначение
Приложение для быстрого создания скриншотов с возможностью выделения области экрана и базовым редактированием прямо перед сохранением.

## Основной сценарий использования
1. Приложение постоянно работает в фоне и отображается в системном трее.
2. Пользователь по одному клику по иконке в трее — запускается процесс создания скриншота.

## Пошаговый процесс создания скриншота

### 1. Затемнение / полупрозрачность экранов
- При запуске захвата все подключенные мониторы становятся полупрозрачными.
- Это визуально показывает пользователю, что включён режим выделения области.

### 2. Выделение области мышью
- Пользователь зажимает левую кнопку мыши и рисует прямоугольную область.
- Область выделяется контрастной рамкой (например, белой или цветной).
- После отпускания кнопки мыши создаётся скриншот выделенной области.

### 3. Редактирование скриншота
После захвата области открывается окно редактора с панелью инструментов:
- **Карандаш** — свободное рисование линий произвольной формы.
- **Текст** — добавление текстовой подписи в произвольном месте.
- **Стрелка** — рисование стрелки для указания на объект.
- **Линия** — рисование прямой линии.

Пользователь может комбинировать инструменты, выделяя и помечая нужные детали на скриншоте. Каждый инструмент можно настраивать: изменять цвет, размер шрифта (для текста) и толщину линий.

### 4. Сохранение результата
- В панели инструментов расположена кнопка сохранения / копирования.
- По клику на неё готовый скриншот копируется в буфер обмена.
- Приложение возвращается в режим ожидания в трее.

## История скриншотов
- Все созданные скриншоты сохраняются в локальную историю.
- История доступна через контекстное меню иконки в трее по правому клику.
- В меню отображается список последних скриншотов с миниатюрами / временем создания.
- При выборе пункта из истории скриншот можно открыть, скопировать повторно или удалить.

## Дополнительные требования
- Поддержка нескольких мониторов.
- Минимальное потребление ресурсов в режиме ожидания.
- Быстрый запуск захвата по двойному клику.
- Интуитивно понятный интерфейс редактора.
